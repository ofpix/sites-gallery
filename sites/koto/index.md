# Koto — Architect-Designed Hotel Cabins, Wellness & Homes

> 英国建筑与设计工作室，专精 **建筑设计的酒店 cabin、水疗养生、私人住宅和室内设计**。每个项目从场地、气候和文化出发，创造与自然融为一体的空间。

---

## 设计风格

- **高端自然风** — 大片留白，大画幅建筑摄影，字体考究，排版呼吸感极强
- **品牌一致性极高** — 从 logo、配色（大地色系/暗绿/灰褐）到文案语调，"calm, intentional, built to perform" 贯穿全线
- **产品线清晰** — CABINS / SPA / HOMES / INTERIORS 四大板块，每个都有独立子站但保持统一视觉
- **引用作为信任信号** — 大量客户/合作伙伴的引用穿插在页面中，用 blockquote 分隔内容区块
- **PDF Lookbook** — 提供年度产品手册下载，传统奢侈品行业的做法
- **移动预览弹窗** — 页面加载后有 Haku（移动酒店 cabin）的宣传弹窗

## 产品线

| 板块 | 说明 |
|------|------|
| **CABINS** | 建筑设计的度假 cabin，可定制 rollout 或 ready-to-order |
| **SPA & WELLNESS** | 水疗空间设计，从 standalone sauna 到完整 wellness suite |
| **HOMES** | 高性能定制住宅 + 标准款 Hytti（单层两/三卧室） |
| **INTERIORS** | "Quiet luxury" 室内全案服务，turnkey 交付 |
| **Haku (新)** | 移动式豪华酒店套房，off-grid 可部署 |

## 设计理念

- "Architecture-first" — 建筑优先，而非产品优先
- "Bespoke to place, built to scale" — 从场地出发，可规模化
- "Performance-led by design" — 性能导向
- 每个设计都回应景观、气候和场地特征
