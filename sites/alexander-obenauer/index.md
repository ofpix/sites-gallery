# Alexander Obenauer — Independent Researcher in Personal Computing

> 独立研究者，专注于**未来个人计算（Personal Computing）方向**——探索可塑软件（Malleable Software）、物品化操作环境（Itemized OS）、时间线组织界面。OLLOS 创造者，Ink & Switch 合作者，手工布面书出版人。

---

## 设计风格

- **极简文人风** — 黑白/浅棕配色，大量留白，纯文本排版。页面结构像一本展开的书。
- **内容即结构** — 没有花哨的 UI 框架，直接通过层级标题和链接组织内容，干净、专注。
- **Lab 一体的发布模式** — 公开文章 + 会员制 Lab Notes，用"Lab"的概念包装整套研究输出
- **布面书质感** — 实体书（Bootstrapping Computing、Augment）用 Buddy Bindery 手工制作，网站延续了同样的考究气质
- **Timeline 概念** — 时间线作为核心组织原则（OLLOS），也反映在站内内容的索引方式上

## 研究焦点

| 项目 | 说明 |
|------|------|
| **OLLOS** | 以时间线为核心组织原则的物品化个人计算环境（2021-2023） |
| **The Interfaces With Which We Think** | 核心研究的多部分介绍 |
| **Embark** | 动态文档 + 计划制定（与 Ink & Switch 合作） |
| **Jasper** | 1987 Canon Cat 用户环境复刻 + 探索 |
| **Item Store** | 物品化用户环境的持久化存储层 |
| **Wonderful** | 为物品化环境设计的新语言 |
| **1997.chat** | AIM 复古重现（疫情期间走红后被收购） |
| **Mail Pilot** | 创新邮件客户端，曾获 Mac App Store #1 |
| **Throttle** | 一次性邮箱地址服务 |
